# fdf
***macOS is only supported***

A simplified graphic representation of a relief landscape
```
$>make
$>./fdf ./test_maps/42.fdf
```
isometric projection: I  
change altitude: < >  
zoom: - +  
translate: ▴ ▾ ◂ ▸  
rotate: W S  
change colors: G V B
